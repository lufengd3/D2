# rax-starter-kit

## Getting Started

### `npm run start`

Runs the app in development mode.

Open [http://localhost:9999](http://localhost:9999) to view it in the browser.

The page will reload if you make edits.

### `npm run lint`

You will see the lint errors in the console.

### `npm run build`
